# Infotecs
Кроссплатформенное приложение с поддержкой Windows (Winsock2) и Linux (POSIX Sockets).

## Структура проекта
- library/ — динамическая библиотека с функциями обработки данных.  
- program1/ — клиент (Producer-Consumer), обрабатывающий ввод пользователя и отправляющий данные.  
- program2/ — TCP-сервер, принимающий данные и проверяющий условия.  
- CMakeLists.txt — файл сборки.
